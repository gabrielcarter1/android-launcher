# Android Launcher Panel (Minimal)

This project provides a **web-based launcher** for Android devices.  
It uses **tel:** links and **intent:** URLs to open:

- Dialer (empty or with number)
- Android Settings (Wi-Fi, Bluetooth, Battery, Security, Location, etc.)
- Installed apps (WhatsApp, Camera, Image Picker)
- Hidden diagnostic / testing menus

## Features

- Minimal design, works on all Android browsers
- No JavaScript required
- Works offline or hosted on GitHub Pages

## Usage

1. Open `launcher.html` in your Android browser.
2. Tap any link to open the corresponding system menu or app.
3. Optionally host on GitHub Pages for web access.

### Hosting on GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages → Branch: main / launcher.html**.
3. Visit `https://<your-username>.github.io/android-launcher/launcher.html` to open the panel in your browser.
